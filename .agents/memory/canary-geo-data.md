# Canary Islands geo data (provinces / islands / municipalities)

Geographic reference tables (`provinces`, `islands`, `municipalities`) are
populated as **data rows**, not generated by code. There is no seed script —
rows live in the database and are exposed read-only via `/api/provinces`,
`/api/islands`, `/api/municipalities` (geo.ts). The mobile app does not use them.

Canonical structure (88 municipalities total):
- Santa Cruz de Tenerife (TF): Tenerife 31, La Palma 14, La Gomera 6, El Hierro 3
- Las Palmas (GC): Gran Canaria 21, Lanzarote 7, Fuerteventura 6

**Why this matters:** if a feature needs a complete municipality list, verify the
DB actually has all 88 — historically only Tenerife's 7 largest were present and
the rest had 0. Backfill with a direct INSERT keyed by `island_id`, guarding
against duplicates; do not expect application code to create them.
